node and npm - overview

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine.
Node.js is used for developing server-side JavaScript applications
The npm is a package manager for Node.js, which makes it quite simple to install additional tools via packages; it comes bundled with Node.js

node and npm - install

Download and install the latest LTS version for your system:
Node.js® and npm

node and npm - verify installation

Open a Terminal and type the commands given bellow
The output should look like the commented lines (after the comment sign '#')

			node -v
			#v12.2.0

			npm -v
			#6.9.0

npm and node - update

If you already have an old version on node, you can update it by:

			# update npm:
			npm install -g npm stable

			#update node:
			npm install -g node

			#check version installation,
			node -v
			npm -v

Create your first Angular App

Create your first Angular App

Create a workspace and init application

Open the terminal in your 'lab1' folder

			# make sure you are in 'lab1' folder by typing:
			pwd

			# create new Angular App (with defaults):
			ng new my-first-app

			# If you do not use git, then use:
			ng new --skipGit=true my-first-app
The Angular CLI installs the necessary Angular npm packages and other dependencies. This can take a few minutes.

Run the app

The Angular CLI includes a server, so that you can easily build and serve your app locally

			# go to my-first-app folder:
			cd my-first-app

			# start the server:
			ng serve --open
The --open option will automatically opens your default browser to http://localhost:4200/
