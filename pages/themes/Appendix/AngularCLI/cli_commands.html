Angular CLI Commands

Run new project


				# ng new ProjectName

				# run teh surver and open the ProjectName in the browser:
				cd ProjectName

Components

Create

					# using full command
					ng generate component my-new-component

					# using the alias
					ng g c my-new-component

					# dry-run: will not make any folder changes, useful to preview.
					ng g c --dry-run test
