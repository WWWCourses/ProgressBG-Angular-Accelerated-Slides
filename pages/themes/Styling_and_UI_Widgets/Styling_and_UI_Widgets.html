Basic styling in Angular

Basic styling in Angular

Global styles

We can style our app using global styles and component styles
Global styles can be included in src/index.html
Usually, we put the global css files into the assets/ folder
We can write and include global css in src/styles.css, as well.
If we have used some third-party styles, which are installed by npm, we can point to them in angular.json file

Component Styles

The styles specified in @Component metadata apply only within the template of that component.
Reference: Component styles @angular guide

Using Bootstrap

Using Bootstrap

Overview

Bootstrap is an open-source HTML, CSS and JS framework that is used to quickly design and customize responsive mobile-first sites.
Among its features are Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful JavaScript plugins
Reference: Introduction @bootsrap docs

Including Bootstrap in Angular App

There are two ways by which we can include Bootstrap into Angular
Using the Bootstrap CDN (Content Delivery Network)
Using npm (node packet manager)

Using Bootstrap by CDN

We can skip the download and link to a Bootstrap CDN to deliver cached version of Bootstrapâ€™s compiled CSS and JS to our project.
Recommended CDN: BootstrapCDN
Reference: BootstrapCDN
Include the Bootstrap CSS and JavaScript(if needed) files in the <head> section of the index.html file of your Angular project with a <link> and <script> tags

Using Bootstrap by CDN


			<!doctype html>
			<html lang="en">
			<head>
				<meta charset="utf-8">
				<title>FormsDemoApp</title>
				<base href="/">
				<meta name="viewport" content="width=device-width, initial-scale=1">
				<link rel="icon" type="image/x-icon" href="favicon.ico">

				<!-- include Bootsrap CSS -->
				<link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.5.0/css/bootstrap.min.css" integrity="sha384-9aIt2nRpC12Uk9gS9baDl411NQApFmC26EwAOH8WgZl5MYYxFfc+NcPb1dKGj7Sk" crossorigin="anonymous">

			</head>
			<body>
				<app-root></app-root>
			</body>
			</html>

Using Bootsrap from npm


			# make sure you are in root app folder and type:
			npm install bootstrap jquery popper.js -save

Your package.json will be update with:


			"dependencies": {
				...
				"bootstrap": "^4.5.0",
				"jquery": "^3.5.1",
				"popper.js": "^1.16.1",
				...
			},

All packages that we've installed are downloaded to the node_modules folder

The CSS and JS must be loaded globally, so we must specify the path to Bootstrap files within the angular.json file in the "styles":[] and the "scripts":[] array in "build" (or "test" ) options:

			...
			"architect": {
				"build": {
					"builder": "@angular-devkit/build-angular:browser",
					"options": {
						"outputPath": "dist/formsDemoApp",
						"index": "src/index.html",
						"main": "src/main.ts",
						"polyfills": "src/polyfills.ts",
						"tsConfig": "tsconfig.app.json",
						"aot": true,
						"assets": [
							"src/favicon.ico",
							"src/assets"
						],
						"styles": [
							"./node_modules/bootstrap/dist/css/bootstrap.css",
							"src/styles.css"

						],
						"scripts": []
					},
					"configurations": {
						"production": {

						}
					}
				},
				...
